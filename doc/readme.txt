/////////////////////////////////////////////////
Eye Texture Generator for VRoid
/////////////////////////////////////////////////
VRoidのデフォルト瞳メッシュを前提とした瞳テクスチャのエディタです

ペイントソフトなしでも簡単にオリジナルの瞳が作れるようにと作成しました

/////////////////////////////////////////////////
使い方
/////////////////////////////////////////////////
-- 左ペイン --
New:新規に瞳を作成します
Open:保存した瞳を呼び出します
Save As:作成中の瞳を保存します

Add:新規にレイヤーを追加します。ボタン横の「Fill」「Linear」でレイヤーの種類を決めてください
Remove:現在のレイヤーを削除します
Duplicate:現在のレイヤーを複製して先頭に追加します
Move Up/Move Down:現在のレイヤーを上または下に移動します
Export:現在の瞳をPNG形式で出力します

-- 右ペイン --
・描画モード
そのレイヤーの描画モードを指定します
「Normal」「Overlay」「HardLight」「SoftLight」「Screen」「Multiply」から選択します
各描画モードはPhotoshopやGIMP等と同じです

・描画位置
そのレイヤーの描画対象を指定します
「Left」で左、「Right」で右、「Both」で両方に描画します

・Fill
ベースの瞳と相似の図形を配置します
Size:レイヤーの大きさを変更します
Fill Color:レイヤーの色を変更します
Blur:レイヤーのぼかしを変更します
X Offset:レイヤーのX位置を移動します
Y Offset:レイヤーのY位置を移動します
Rotate：レイヤーを回転させます
Cutoff：レイヤー中央をくりぬいてリング状のレイヤーにします

・Linear
Top:レイヤーの上端を変更します
Bottom:レイヤーの下端を変更します
Fill Color:レイヤーの色を変更します
Blur:レイヤーのぼかしを変更します
Start Offset:目の外側位置の高さを変更します
End Offset:目の外側位置の高さを変更します
